# Projet GLP n 49 - ECRITURE

## Nom du projet

Logiciel de reconnaissance de caracteres et de mots manuscrits

## Objectif du site

Ce site web statique presente le projet de Genie Logiciel et Projet (GLP) n 49 intitule ECRITURE.
Il a ete realise dans le cadre d'un TD universitaire portant sur le developpement web (HTML5 / CSS3).

Le site decrit les objectifs, les specifications, les technologies utilisees, l'equipe et
l'avancement du projet de reconnaissance d'ecriture manuscrite.

## Composition de l'equipe

Le projet est realise par un trinome d'etudiants :

| Membre  | Role principal                    |
|---------|-----------------------------------|
|  Yasmine| Responsable IA et Donnees         |
|  Rhal   | Responsable Algorithmique         |
|  Suha   | Responsable IHM (Interface)       |


## Technologies utilisees

### Pour le logiciel GLP :
- **Java** (JDK 17+) - Langage principal
- **Java Swing** - Interface graphique
- **Matrices binaires** - Representation des caracteres
- **IA simplifiee** - Apprentissage adaptatif

### Pour le site web :
- **HTML5** - Structure semantique
- **CSS3** - Mise en forme et design

## Structure du site

```
|-- img/

+-- readme.md               (Ce fichier)
```

## Instructions d'ouverture du site

1. Telecharger l'ensemble des fichiers du dossier `site-glp-ecriture`
2. Conserver la structure des dossiers intacte (css/, img/)
3. Ouvrir le fichier `index.html` dans un navigateur web moderne
   (Firefox, Chrome, Edge, Safari)
4. Naviguer entre les pages via le menu horizontal en haut de chaque page

### Validation W3C

- Le code HTML est conforme aux normes HTML5 du W3C
- Le code CSS est conforme aux normes CSS3 du W3C
- Validation possible sur : https://validator.w3.org/
