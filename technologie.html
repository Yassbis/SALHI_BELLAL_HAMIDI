Java – Langage principal

Le projet est entierement developpe en Java, un langage de programmation oriente objet, portable et robuste. Java est le langage impose dans le cadre du module GLP.

  • Langage oriente objet, facilitant la modularite du code
  • Portabilite grace a la JVM (Java Virtual Machine)
  • Riche ecosysteme de bibliotheques standard
  • Gestion automatique de la memoire (Garbage Collector)

A savoir

Java est l'un des langages les plus utilises dans l'enseignement universitaire en informatique. Sa syntaxe stricte et son typage fort en font un excellent outil pedagogique pour apprendre les concepts du genie logiciel.

IHM graphique : Java Swing

L'interface graphique du logiciel est realisee avec la bibliotheque Java Swing, qui offre un ensemble complet de composants pour creer des interfaces de bureau.

  • JFrame : Fenetre principale de l'application
  • JPanel : Panneau de dessin pour la grille de saisie
  • MouseListener / MouseMotionListener : Capture des evenements souris
  • Graphics2D : Rendu graphique du trace et de la grille
  • JButton, JLabel, JTextField : Composants d'interface

Pourquoi Swing ?

Swing est integre nativement dans le JDK Java, ce qui evite toute dependance externe. Il est particulierement adapte aux projets universitaires car il ne necessite aucune installation supplementaire.

Algorithmes : Traitement matriciel

Le coeur algorithmique du projet repose sur le traitement et la comparaison de matrices binaires representant les caracteres manuscrits.

  • Discretisation du trace en grille 15 x 15
  • Conversion en matrice binaire (0 et 1)
  • Calcul de distance entre matrices (similarite)
  • Recherche du caractere le plus proche dans la base
  • Assemblage sequentiel pour la reconnaissance de mots

Detail technique

La mesure de similarite utilisee peut etre une distance de Hamming (nombre de bits differents entre deux matrices) ou une distance euclidienne adaptee. Le choix de la metrique influence directement la precision de la reconnaissance.

IA simplifiee : Apprentissage adaptatif

Le module d'intelligence artificielle est simplifie et simule, concu pour introduire les concepts de base de maniere pedagogique.

  • Base de donnees individuelle par utilisateur
  • Apprentissage incremental a chaque nouvelle saisie
  • Comparaison personnalisee avec le style calligraphique de l'utilisateur
  • Verification des mots via dictionnaire externe (fichiers CSV/TXT)
  • Correction automatique basee sur la base de connaissances

Limites de l'approche

Cette IA simplifiee ne peut pas generaliser a tous les utilisateurs ni a toutes les ecritures possibles. Elle est concue pour apprendre le style specifique d'un seul utilisateur et s'ameliorer progressivement dans ce cadre restreint.
